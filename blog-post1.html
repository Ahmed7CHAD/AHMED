Découverte HTML & CSS

J'ai découvert à quoi ressemble le code des sites internet : le HTML et le CSS ! Ce sont les langages de programmation qui permettent qu'un site s'affiche, comme celui que vous voyez en ce moment.

  • Le HTML (HyperText Markup Language) est un sorte de code , un langage de balisage . Il permet de créer et structurer des pages Web.
  • Les CSS, Cascading Style Sheets (feuilles de styles en cascade), servent à mettre en forme des documents web, type page HTML ou XML.
